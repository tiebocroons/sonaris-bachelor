# Sonaris Bachelor - Audiogram Scanner 👋

A React Native Expo app for scanning and analyzing audiograms with professional-grade code analysis and remediation powered by the React Native MCP Server.

## Features

- 📱 Multi-screen audiogram scanning interface
- 🔧 Expert code analysis and automatic remediation
- ✅ Automated testing and quality assurance
- 🛡️ Security vulnerability detection and fixing
- ⚡ Performance optimization with memory leak prevention
- ♿ Accessibility compliance checking

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.

## Development Tools - React Native MCP Server

This project includes the **React Native MCP Server** (`@mrnitro360/react-native-mcp-guide`), an expert-level development companion providing:

### 🔧 Features
- **Expert Code Remediation** - Automatic fixing of security, performance, and quality issues
- **Advanced Refactoring** - Component modernization with test generation
- **Security Auditing** - Vulnerability detection and automatic remediation
- **Performance Analysis** - Memory optimization and rendering improvements
- **Automated Testing** - Industry-standard test suite generation
- **Accessibility Compliance** - WCAG 2.1 AA standard validation

### Quick Commands

```bash
# Automatic code remediation with expert-level fixes
claude "remediate_code with remediation_level='expert'"

# Refactor a component with tests
claude "refactor_component with refactor_type='comprehensive' and include_tests=true"

# Security audit with auto-fix
claude "analyze_codebase_comprehensive with analysis_types=['security']"

# Performance analysis
claude "analyze_codebase_performance"

# Generate tests for a component
claude "generate_component_test with component_name='HomeScreen'"

# Check dependencies for vulnerabilities
claude "audit_packages with auto_fix=true"
```
